# DropdownButton and IconButton Demo

A simple Flutter application demonstrating the use of DropdownButton and IconButton widgets.

## Run

flutter pub get

flutter run

## Widgets Demonstrated

### DropdownButton
1. value
2. items
3. onChanged

### IconButton
1. icon
2. iconSize
3. color
